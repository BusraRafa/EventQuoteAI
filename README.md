# 🎉 EventQuoteAI

EventQuoteAI is a Python-based AI project that generates event quotations using large language models.  
It explores how AI can automate event planning and pricing suggestions based on user inputs.

---

## 🚀 Overview

This project is an experimental implementation of an AI-powered quote generator for events such as:

- Weddings  
- Corporate events  
- Parties  
- Conferences  

Users provide event details, and the system generates structured quote responses.

---

## 📂 Project Structure
```
EventQuoteAI/
│── app.py # Main application script
│── new_app.py # Updated/experimental version of the app
│── radu.py # Testing / alternative logic
```

---

## 🧠 How It Works

1. User inputs event details (type, size, requirements)  
2. The app sends the data to an AI model  
3. AI generates:
   - Event breakdown  
   - Structured quote output  

---

## 🛠️ Tech Stack

- Python 🐍  
- OpenAI API (or similar LLM)  
- Prompt Engineering

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/BusraRafa/EventQuoteAI.git
cd EventQuoteAI
```
2. Install dependencies:
```
pip install -r requirements.txt
```
3. Set environment variables:
```
OPENAI_API_KEY=your_api_key_here
```
